<h1>Note Taker</h1>
<p>A website that can be used for writeing notes or creating tasks. It was created with Node.js and Express
     https://ancient-scrubland-73509.herokuapp.com/
</p>
 <h3>User Story</h3>
 <ul>
       <li>AS A user, I want to be able to write and save note</li>
       <li>I WANT to be able to delete notes I've written before</li>
       <li>SO THAT I can organize my thoughts and keep track of tasks I need to complete</li>
</ul>
